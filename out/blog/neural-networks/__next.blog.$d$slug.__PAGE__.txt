<h3><strong>What is a Neural Network?</strong></h3>
<p>A neural network is a type of machine learning model that is inspired by how the human brain works. It is made up of layers of <strong>neurons</strong> (or nodes), each connected to others by <strong>weights</strong>. These connections help the network learn patterns in the data.</p>
<p>A basic neural network has:</p>
<ol>
<li><strong>Input Layer</strong>: Where the data comes in.</li>
<li><strong>Hidden Layer</strong>: Where the data is processed.</li>
<li><strong>Output Layer</strong>: Where the result or prediction comes out.</li>
</ol>
<p><img src="/neuralnetwork.png" alt="Neural Network"></p>
<h3><strong>How It Works</strong></h3>
<ul>
<li>Each neuron in a layer gets input, multiplies it by a weight, and adds a bias.</li>
<li>The result is passed through an <strong>activation function</strong> (like Sigmoid) to get the output.</li>
<li>The neural network adjusts the weights and biases to make the output closer to the desired result (this process is called <strong>training</strong>).</li>
</ul>

<ol>
<li>We calculate the weighted sum of inputs for the hidden layer.</li>
<li>Pass this through a Sigmoid function to get the activation (output) of the hidden layer.</li>
<li>Repeat for the output layer to get the final prediction.</li>
</ol>

<h3><strong>What happens in Backpropagation?</strong></h3>
<ol>
<li>We calculate how much the output error is influenced by each weight.</li>
<li>We adjust the weights and biases in the direction that reduces the error.</li>
</ol>
<hr>
<h2><strong>Step 4: Training the Network</strong></h2>
<p>Now, we train the network! We will run <strong>forward propagation</strong> and <strong>backpropagation</strong> for many iterations (epochs) to improve the model's predictions.</p>

<span data-line=""><span style="--shiki-light:#24292EFF;--shiki-dark:#B392F0">        output </span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#24292EFF;--shiki-dark:#B392F0"> self</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">.</span><span style="--shiki-light:#6F42C1;--shiki-dark:#B392F0">forward</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">(X)</span><span style="--shiki-light:#C2C3C5;--shiki-dark:#6B737C">  # Get the predicted output</span></span>

<h3><strong>Training Steps:</strong></h3>
<ol>
<li><strong>Forward pass</strong>: Get predictions.</li>
<li><strong>Backward pass</strong>: Adjust weights and biases.</li>
<li>Repeat for many epochs (iterations) to make the model better.</li>
</ol>
