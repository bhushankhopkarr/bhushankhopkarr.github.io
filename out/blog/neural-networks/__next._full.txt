12:T8cc2,<p>We’ll build a very basic neural network using Python, without using any deep learning libraries. We'll also walk through the basic ideas of how neural networks work and create a simple neural network that learns to solve a small problem (XOR).</p>
<h3><strong>What is a Neural Network?</strong></h3>
<p>A neural network is a type of machine learning model that is inspired by how the human brain works. It is made up of layers of <strong>neurons</strong> (or nodes), each connected to others by <strong>weights</strong>. These connections help the network learn patterns in the data.</p>
<p>A basic neural network has:</p>
<ol>
<li><strong>Input Layer</strong>: Where the data comes in.</li>
<li><strong>Hidden Layer</strong>: Where the data is processed.</li>
<li><strong>Output Layer</strong>: Where the result or prediction comes out.</li>
</ol>
<p><img src="/neuralnetwork.png" alt="Neural Network"></p>
<h3><strong>How It Works</strong></h3>
<ul>
<li>Each neuron in a layer gets input, multiplies it by a weight, and adds a bias.</li>
<li>The result is passed through an <strong>activation function</strong> (like Sigmoid) to get the output.</li>
<li>The neural network adjusts the weights and biases to make the output closer to the desired result (this process is called <strong>training</strong>).</li>
</ul>
<hr>

<ul>
<li><strong>Input Layer</strong>: 2 nodes (because we'll use two input features)</li>
<li><strong>Hidden Layer</strong>: 4 nodes (this can be adjusted)</li>
<li><strong>Output Layer</strong>: 1 node (since we’re solving a binary problem like XOR)</li>
</ul>
<p>We'll also initialize <strong>weights</strong> (how strongly each input influences the output) and <strong>biases</strong> (to shift the activation function).</p>

<p>In forward propagation:</p>
<ol>
<li>We calculate the weighted sum of inputs for the hidden layer.</li>
<li>Pass this through a Sigmoid function to get the activation (output) of the hidden layer.</li>
<li>Repeat for the output layer to get the final prediction.</li>
</ol>
<hr>
<h2><strong>Step 3: Backpropagation</strong></h2>
<p>Backpropagation is how the neural network learns. It adjusts the weights to make the predictions more accurate.</p>
<p>Here, we use <strong>Mean Squared Error</strong> (MSE) to measure the difference between the predicted output and the actual result. Then, we update the weights using the <strong>gradients</strong> (how much the weights affect the output).</p>

<h3><strong>What happens in Backpropagation?</strong></h3>
<ol>
<li>We calculate how much the output error is influenced by each weight.</li>
<li>We adjust the weights and biases in the direction that reduces the error.</li>
</ol>

<span data-line=""><span style="--shiki-light:#24292EFF;--shiki-dark:#B392F0">nn </span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#6F42C1;--shiki-dark:#B392F0"> NeuralNetwork</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">(input_size</span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#1976D2;--shiki-dark:#F8F8F8">2</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">, hidden_size</span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#1976D2;--shiki-dark:#F8F8F8">4</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">, output_size</span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#1976D2;--shiki-dark:#F8F8F8">1</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">)</span></span>
<span data-line=""> </span>
<span data-line=""><span style="--shiki-light:#C2C3C5;--shiki-dark:#6B737C"># Train the neural network</span></span>
<span data-line=""><span style="--shiki-light:#24292EFF;--shiki-dark:#B392F0">nn</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">.</span><span style="--shiki-light:#6F42C1;--shiki-dark:#B392F0">train</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">(X, y, epochs</span><span style="--shiki-light:#D32F2F;--shiki-dark:#F97583">=</span><span style="--shiki-light:#1976D2;--shiki-dark:#F8F8F8">10000</span><span style="--shiki-light:#212121;--shiki-dark:#BBBBBB">)</span></span></code></pre></figure>
<p>In this code:</p>
<ul>
<li><code>X</code> is the input (two features).</li>
<li><code>y</code> is the output (XOR values).</li>
</ul>
<p>After training, the network will learn how to predict XOR outputs based on the inputs.</p>
