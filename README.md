# bhushankhopkarr.github.io

Personal portfolio and blog — built with [Next.js](https://nextjs.org), [Tailwind CSS](https://tailwindcss.com), and [shadcn/ui](https://ui.shadcn.com).

[![Linear](https://img.shields.io/badge/Tracked%20on-Linear-5E6AD2?logo=linear&logoColor=white)](https://linear.app)

## Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS + shadcn/ui
- **Animations**: Framer Motion + Magic UI
- **Blog**: MDX with rehype-pretty-code
- **Deployment**: GitHub Pages

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Project Management

This project is tracked on [Linear](https://linear.app). Issues and feature requests follow the templates in [`.github/ISSUE_TEMPLATE`](.github/ISSUE_TEMPLATE). When opening a pull request, please reference the corresponding Linear issue ID (e.g. `LIN-123`) in the PR description.
